공공데이터 RESTful API를 활용해 경기도 공공 도서관 정보를 조회하는 웹 애플리케이션을 구현했습니다.<br/>
fetch를 사용해 데이터를 비동기로 가져오고, API 응답 데이터를 원본 상태로 관리, 검색/지역 필터링/정렬/더보기 페이지네이션을 파생 상태로 분리해 처리했습니다.<br/>
검색 입력에는 debounce를 적용해 UX를 개선했으며, API 키는 환경변수로 관리해 코드에 직접 노출되지 않도록 구성했습니다.<br/>
또한 GSAP을 활용해 리스트 및 UI 요소에 자연스러운 등장 애니메이션을 적용해 사용자 경험을 개선했습니다.
